## Repo layout

```
.
├── docs/                     # the canonical Markdown guide (deployed via GitHub Pages)
│   └── README.md             # main entry — 15 chapters, snippets, references
├── app/                      # optional Next.js scaffold for an interactive site
├── DEPLOYMENT.md             # GitHub Pages + CI setup walkthrough
└── .github/workflows/        # CI and deploy
```

## Quick start

**Just want to read it?** Open `docs/README.md` here on GitHub — collapsibles work in the renderer.

**Want to publish it on GitHub Pages?** See [`DEPLOYMENT.md`](./DEPLOYMENT.md). For pure-Markdown deploy, you're 3 settings clicks away from a live URL.

**Want the interactive Next.js site?**
```bash
cd app
nvm use && npm install
npm run dev
```
